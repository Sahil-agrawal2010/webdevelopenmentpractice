What is React?
React is a javascript library that is used to build dynamic and interactive user interfaces.
It was developed by Facebook in 2011/2012.
It is currently the most widely used javascript library for frontend developement.
React works on single page application (SPA) format.

Problems with javascript:-

iv) Javascript is hard to maintain.
